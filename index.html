Haskell Optimization Handbook¶

This book assumes you are using ghc 8.10.x or above and using the Linux operating system. If you are new please start with the How to use this book chapter. If you are returning then feel free to jump to the Table of Contents and best wishes. It is dangerous to go alone, please take this handbook and good luck!
